How It Works?

Google Forms

Google Forms serves as the interface for attendees to confirm their presence. Each member has a personalized form linked to their unique QR code. Upon scanning, attendees fill out these forms, updating attendance data instantly in the associated Google Sheet.

Google Sheets

Google Sheets acts as the central database for attendance records. Data collected from Google Forms submissions is stored in Google Sheets, allowing easy access, organization, and analysis.

Google App Scripts

Google Apps Script automates processes within the attendance system. Through custom scripts, it triggers email notifications to parents upon attendance confirmation, ensuring timely communication.

QR Code

The QR code generator creates unique identifiers for each attendee. These QR codes, containing member-specific information, facilitate quick and accurate attendance tracking.

JavaScript

JavaScript enhances the interactivity and responsiveness of the attendance system. It powers dynamic elements on the website, such as QR code generation and form submission handling.

HTML

HTML provides the structure and layout of the attendance system's web interface. It defines the elements and content displayed to users, including QR code representations and form layouts.

CSS

CSS enhances the visual presentation and styling of the attendance system's web interface. It defines the appearance of elements, such as colors, fonts, and spacing, ensuring a cohesive and appealing design.
